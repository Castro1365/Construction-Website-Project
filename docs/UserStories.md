# User Stories

### US1 – Browse Services
> As a **homeowner**, I want to view a list of construction services so that I can check if the company can renovate my house.

**Acceptance Criteria**
- Services page lists at least 5 services.
- Each service has a short description.

---

### US2 – Learn About the Company
> As a **new customer**, I want to read about the company’s background so that I can decide if I trust them.

**Acceptance Criteria**
- About page exists.
- Contains company history, team/values, and why choose us.

---

### US3 – Request a Quote
> As a **potential client**, I want to submit my contact information and project details so that the company can get back to me.

**Acceptance Criteria**
- Contact page contains a form with name, email, service/project type, and message.
- Clicking “Submit” shows a confirmation message in the browser (no real email required).

---

### US4 – Navigate Easily
> As a **site visitor**, I want a clear navigation menu so that I can move between pages without getting lost.

**Acceptance Criteria**
- All pages show the same navigation bar.
- Navbar links jump to the correct pages.

---

### US5 – View on Different Devices
> As a **mobile or desktop user**, I want the pages to remain readable so that I can use the site from any device.

**Acceptance Criteria**
- Text is readable and layout doesn’t break on smaller windows.
- No horizontal scrolling required for main content.
