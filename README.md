This project is a simple **Construction Company Website** built as part of the Software Engineering final group assignment.

- Personas  
- User stories  
- Architecture design  
- Wireframes  
- Website implementation (HTML/CSS/JS)  
- Version control using Git & GitHub

All development happens on your own branch → then merged into **main**.
